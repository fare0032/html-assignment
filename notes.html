5 HTML tags I've learned and liked

    The Unordered List:
    This tag lists items with small bullet points by default
    The Image tag:
    This tag improves the deisgn and appearance of your web page
    The Color tag:
    This tag can allow you to change the color of your text and your web page background
    The Italic tag:
    This tag defines a part of text in an alternate voice or mood
    The Style tag:
    This tag allows you to choose a color for your paragraphs and your fonts
